Frequently Asked Questions

How do I get started with Orchestral?

Sign up for a free account, generate your API key from the dashboard, and follow our quickstart guide to make your first API call.

What models are supported?

We support all major models from OpenAI (GPT-4, GPT-3.5), Anthropic (Claude 3), and Google (Gemini Pro).

How is billing calculated?

Billing is based on your subscription tier and usage. Each tier includes a set number of API calls and tokens per month.

Can I switch between models?

Yes! You can specify which model to use for each request, or let our intelligent routing automatically select the best model.

What's the uptime SLA?

We offer 99.9% uptime for Starter plans and above, with 99.99% for Enterprise customers.
