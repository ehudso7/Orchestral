Orchestral Documentation

Welcome to Orchestral, the unified API platform for AI model orchestration. Access multiple AI providers through a single API.

New to Orchestral? Start with our Quick Start Guide to make your first API call in minutes.

Quick Start

1. Get Your API Key

Sign up for an account and generate an API key from your dashboard.

Get API Key

2. Install the SDK

# Python pip install orchestral-ai # Node.js npm install orchestral # Go go get github.com/orchestral/orchestral-go

3. Make Your First Request

import orchestral

client = orchestral.Client(api_key="your-api-key")

response = client.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "Hello, world!"}
    ]
)

print(response.content)

Authentication

All API requests require authentication using an API key. Include your API key in the request headers:

X-Api-Key: orch_your_api_key_here

Security: Never expose your API key in client-side code or public repositories.

Completions API

POST /v1/completions

Generate completions using any supported model.

Request Body

{
  "model": "gpt-4o",
  "messages": [
    {
      "role": "system",
      "content": "You are a helpful assistant."
    },
    {
      "role": "user",
      "content": "Explain quantum computing"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 1000
}

Response

{
  "id": "cmpl-123456",
  "model": "gpt-4o",
  "content": "Quantum computing is a revolutionary...",
  "usage": {
    "prompt_tokens": 20,
    "completion_tokens": 150,
    "total_tokens": 170
  },
  "created": 1701234567
}
